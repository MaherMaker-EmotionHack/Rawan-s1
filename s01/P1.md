# S01-P1

Consider the following code :

```cpp
#include <iostream>
using std::cout, std::endl;


class Rectangle {
    double height;
    double width;

public:
    void changeDimensions(double h, double w) {
        height = h;
    }
    double surface() {
        return height*width;
    }
    void print() const {
        cout << "Rectangle[" << width << "x" << height << "], surface=" << surface() << endl;
    }
};


int main() {
    Rectangle r1;
    r1.changeDimensions(4, 12);
    r1.print();
}
```

There are two problems with the code, one preventing compilation and another one which prevents the code to behave as expected.
Find them, and correct the code **without removing anything**.
